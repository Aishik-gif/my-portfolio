PayMe is a Wallet web app made using React, Javascript, MongoDB, Express and Node.js.

## Tech Stack

- React
- Javascript
- MongoDB (Mongoose)
- Express
- Node.js

## Features

1. Authentication for users done using JWT.
2. Get a random amount of money on signing up.
3. Send or receive money to any other user on the app.
